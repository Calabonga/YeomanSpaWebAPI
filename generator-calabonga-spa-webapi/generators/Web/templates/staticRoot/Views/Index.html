SPA & WebAPI

This is a clear Wep Api application base on OWIN middleware. This is application build without ASP.NET MVC.

version 0.1.0

This application template was created for Single Page Application (SPA) application. This is the version 1.0.0 of the solution.

The JavaScript frameworks that you can use to create SPA is:

  • Angular JS
  • Aurelia (Old version Durandal JS)
  • Breeze JS
  • HotTowel
  • Meteor
  • Ember
  • Backbone JS
